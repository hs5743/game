# 雙語字卡學習村 RPG

這是一個可繼續擴充的 2D 網頁 RPG 原型。主題改為「24 張雙語學習字卡（12 正向品格 × 12 情緒詞彙）」，學生可以在村莊、學院、字卡館、情緒花園、森林、池畔與農田之間移動，收集字卡並完成英文詞彙挑戰。

## 目前內容

- 7 個連接地圖：學習村廣場、北村學院、雙語字卡館、南方情緒花園、晨光森林、心情池畔、合作農田
- 24 張字卡任務：12 張正向品格、12 張情緒詞彙
- 使用字卡圖片作為遊戲任務與右側圖鑑預覽
- 玩家姓名輸入：NPC 對話會稱呼玩家姓名
- 情境應用題：題目改為配合卡片主題的生活情境，不只是單字選擇
- 字卡圖鑑與學習筆記整合：已收集字卡會顯示縮圖，點開可查看圖片、情境題、正確答案、提示與生活任務
- 3 條跨地圖 NPC 小任務：村長、森林守望者、農場老師會請學生到其他地圖找 NPC 回報，完成後增加勇氣心
- 分類進度：分開顯示 12 張品格卡與 12 張情緒卡完成度
- 平板觸控操作：iPad / 平板會顯示方向鍵與互動鍵
- 遊戲式 HUD：重要資訊以小圖形化按鈕顯示，詳細資訊用可關閉視框呈現
- 重新開始會清除玩家姓名與所有進度，回到開始畫面
- 自動換地圖的發光出口
- NPC 對話、問答挑戰、收集進度、勇氣心、完成畫面
- 純 HTML/CSS/JavaScript，可部署到 GitHub Pages

## 操作

- 方向鍵或 WASD：移動
- Space：對話、挑戰字卡
- 走上發光出口：切換地圖
- 圖鑑：已收集字卡可點縮圖查看完整學習筆記，未收集字卡維持鎖定

## 檔案結構

- `index.html`：遊戲畫面與 HUD
- `styles.css`：版面、問答面板、字卡圖鑑樣式
- `card-art.js`：GitHub Pages 可直接載入的字卡圖片資料
- `game.js`：地圖、角色、字卡資料、問答與遊戲邏輯
- `assets/cards/`：已壓縮成網頁用大小的 24 張字卡圖片

## 擴充方式

在 `game.js` 中可以繼續擴充：

- `cards`：字卡資料，例如中文、英文、角色名、提示語
- `quizBank`：題目生成方式
- `worldMaps`：地圖、出口、NPC、字卡挑戰位置
- `drawTile`：新增地形，例如商店、橋、校舍、任務告示板

建議下一步可以加：

- 主線任務：村長請學生找回散落的 24 張字卡
- 小任務：可繼續擴充更多 NPC 跨地圖探索、撿拾物品、回報任務
- 背包：收集星星、種子、徽章
- 學習紀錄：完成哪些字卡、錯過哪些單字
- 分章節地圖：品格村、情緒森林、合作農場、勇氣山路

## 本機預覽

直接開啟 `index.html` 可以玩。若瀏覽器因安全設定擋圖片，可在資料夾中啟動簡單伺服器：

```bash
python -m http.server 8080
```

再開啟：

```text
http://localhost:8080
```

## GitHub Pages

部署到 GitHub Pages 時，Repository Settings -> Pages，選擇 `main` 分支與根目錄即可。更新後若看到舊版，請強制重新整理或在網址後加上 `?v=2`。
